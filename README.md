Современное веб-приложение для управления задачами, разработанное на TypeScript с использованием Vite и SCSS.

## Демо

![Image alt](https://github.com/siega52/Task-To-Do/raw/main/src/ToDo.jpg)

## Возможности

- **Добавление, редактирование и удаление задач**
- **Система приоритетов** (низкий, средний, высокий)
- **Категории задач** для удобной организации
- **Фильтрация задач** (все, активные, завершенные)
- **Статистика выполнения** в реальном времени
- **Темная/светлая тема** с сохранением настроек
- **Автосохранение** в Local Storage
- **Адаптивный дизайн** для мобильных устройств
- **Сортировка задач** (по дате, приоритету)

## Технологии

- **TypeScript** - строгая типизация и надежность
- **Vite** - современный сборщик для быстрой разработки
- **SCSS** - препроцессор для продвинутых стилей
- **Local Storage API** - сохранение данных в браузере
- **CSS Variables** - динамические темы оформления
